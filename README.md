# 🎵 SoundWave — Django Music App

Аналог Spotify на Django с JSON-базой данных.

## Быстрый старт

```bash
# 1. Установи зависимости
pip install django

# 2. Перейди в папку проекта
cd soundwave

# 3. Примени миграции (создаёт db.sqlite3 для сессий/авторизации)
python manage.py migrate

# 4. Запусти сервер
python manage.py runserver

# 5. Открой в браузере
# http://127.0.0.1:8000
```

## Возможности

- 🎵 **Загрузка треков** — `/upload/` — MP3, WAV, OGG
- 🔥 **Карусель топ треков** — сортировка по прослушиваниям
- ⭐ **Топ исполнителей** — сумма прослушиваний всех треков автора
- 📋 **Список всех треков** — с сортировкой по популярности
- ▶️ **Встроенный плеер** — внизу страницы, с переключением треков
- 🎤 **Страница исполнителя** — все треки автора

## Структура

```
soundwave/
├── config/          — настройки Django
├── music/           — приложение
│   ├── models.py    — работа с JSON базой данных
│   ├── views.py     — страницы
│   └── urls.py      — маршруты
├── templates/       — HTML шаблоны
├── static/          — CSS и JS
├── media/tracks/    — загруженные аудиофайлы
└── data/
    └── database.json  ← ВСЯ БАЗА ДАННЫХ ЗДЕСЬ
```

## База данных (JSON)

Файл `data/database.json` содержит все треки и исполнителей.
Структура понятна без дополнительных пояснений.

## API

- `GET /api/tracks/` — все треки в JSON
- `POST /play/<id>/` — увеличить счётчик прослушиваний
